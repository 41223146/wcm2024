# 如何利用 Codespaces 維護網站
採用 Github Codespaces 維護網站的好處是, Visual Studio code 介面提供多終端機與多 ports 開啟功能
可使用init_replit指令安裝Python模組 chmod u+x init_replit

./init_replit

接著輸入(這是動態網站)

python3 main.py

再開啟新的終端機

python3 -m http.server

這是靜態網站

即可編輯網站
